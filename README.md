## e-school

School management (report card generation, absence and payment tracking, notification system and more...)

### Stack 📚

- Bun
- Prisma
- PostgreSQL

### Deploy 🚀

```shell
https://e-school-a85e.onrender.com/
```

### Endpoints 📡

```shell
/swagger
```

### Installs 🛠️

```shell
bun install
```

### Run 🏃

```shell
bunx prisma db push
bun dev
```
